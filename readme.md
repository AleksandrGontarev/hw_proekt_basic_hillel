# Критерии:
# 1 Выполнена верстка всех страниц и всех блоков на этих страницах. (минимум 2 стр. Главная и О нас)
# 2 Проект полностью соответствует техническому заданию
# 3 Весь код написан в одной стилистике (один тип кавычек, одинаковые отступы между блоками свойств…)
# 4 Главная страница имеет название index.html.
# 5 В названиях и расширениях файлов нет заглавных букв и пробелов, использованы только латинские символы (касается всех файлов проекта)
# 6 Верстка проходит валидацию без ошибок, предупреждения – допустимы
# 7 Верстка проходит тестирование на ПП с расхождением не более 5 пикселей
# 8 Вёрстка идентично отображается в последних версиях браузеров Chrome и Firefox
# 9 Телефоны размечены ссылками с протоколом tel
# 10 Email размечены ссылками с протоколом mailto
# 11 Шрифты подключены локально и находятся в папке fonts.
# 12 Указаны альтернативные шрифты
# 13 Выполнена предзагрузка шрифтов
# 14 Названия полей форм привязаны к своим полям с помощью <label>
# 15 У контентных изображений в <img> заполнен alt
# 16 Не используются дробные значения для px
# 17 Сделаны микроанимации для интерактивных элементов
# 18 Все элементы имеют все состояние указанные в стайлгайде, то, что в стайлгайде не указано делать не нужно, но можно 
# сделать на свое усмотрение.
# 19 Отсутствует транслит в названиях классов.
# 20 Уровень вложенность селекторов не больше 3-х
# 21 В коде не используется !important
# 22 В коде нет селекторов по идентификатору и по тегу (кроме тега «а»)
# 23 Нет глобальных изменений стилей (глобально изменить допускается только ссылки, состояние ссылок по умолчанию 
# прорисовано в стайлгайде, a також box-sizing: border-box;  margin: 0;)
# 24 Для блока, у которого есть фоновое изображение, прописан фоновый цвет, который соответствует преобладающему цвету 
# изображения
# 25 Явно указано подходящее vertical-align для потоковых блоков с display: inline-block
# 26 Все интерактивные элементы доступны с клавиатуры
# 27 Все интерактивные элементы без контента имеют текстовое описание (aria-label)
# 28 Использован препроцессор sass, синтаксис scss (если проект на гапле)
# 29 Все цвета из макета вынесены в переменные препроцессора / css
# 30 Используется методология БЭМ (только для нейминга классов)
# 31 Каждый БЭМ блок описан в своем файле (+ файл для подключения шрифтов, + для переменных)
# 32 Верстка проходит тестирование на переполнение контентом (если в каком-то блоке добавиться / уменьшить текст – верстка # не должна разваливаться)
# 33 Выполнена ретинизация растровой графики (вся растровая графика подключенная через стили должна быть адаптирована для 
# ретина экранов)
# 34 Выполнено кадрирование контентных изображений. (все контентные изображения которые изменяются свой размер между
# версиями экранов должны быть кадрированы)
# 35 Вся графика ужата, стили минифицированы, html ужат
# 36 Верстка не разваливается за пределами точек адаптивности, в том числе верстка не разваливается если экран меньше или # больше минимальной/максимальной ширины сайта
# 37 Проект залит на гитхаб
# 38 На гит не попадает собранная версия проекта, папка с модулями и файлы локальных настроек
# 39 Сборка проекта запускается командами gulp и gulp build
